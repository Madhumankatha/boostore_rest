# boostore_rest
Using fastapi to create rest service

<h3>INTALLATION</h3>
<pre>
pip install fastapi
pip install uvicorn
</pre>

<h3>COMMANDS</h3>
<pre>
<h5>TO RUN SERVER</h5>
<p>uvicorn app:app --reload</p>
<h5>Appilication Endpoint: </h5>
<p> http://127.0.0.1:8000</p>
</pre>
